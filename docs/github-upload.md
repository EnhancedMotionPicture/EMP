# GitHub upload

## Repo name
Use `emp-mov`

## Web UI upload
1. Create a new public GitHub repository.
2. Do not initialize it with README, .gitignore, or license if uploading the project contents directly.
3. Unzip the upload package.
4. Open the unzipped folder.
5. Drag the contents into GitHub's web uploader.
6. Commit with: `Initial EMP.MOV open-source release`
